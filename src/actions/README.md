# Server Actions

Server Actions de Next.js para operaciones del servidor:

## Auth
- `login.ts` - Login de usuario
- `register.ts` - Registro de usuario
- `logout.ts` - Logout
- `verify-phone.ts` - Verificación de teléfono

## Tasks
- `get-today-tasks.ts` - Obtener tareas del día
- `complete-task.ts` - Completar tarea
- `create-custom-task.ts` - Crear tarea personalizada

## Store
- `get-items.ts` - Obtener items de la tienda
- `purchase-item.ts` - Comprar item

## Inventory
- `get-inventory.ts` - Obtener inventario
- `equip-item.ts` - Equipar item

## Profile
- `get-profile.ts` - Obtener perfil
- `update-profile.ts` - Actualizar perfil

## Payments
- `create-checkout.ts` - Crear sesión de checkout
- `create-portal-session.ts` - Crear sesión del portal de clientes
