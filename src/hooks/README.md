# Custom React Hooks

Hooks personalizados:

- `use-avatar-state.ts` - Estado del avatar
- `use-realtime-avatar.ts` - Avatar en tiempo real (Supabase Realtime)
- `use-realtime-wallet.ts` - Wallet en tiempo real
- `use-countdown.ts` - Cuenta regresiva
- `use-timer.ts` - Temporizador
- `use-media-query.ts` - Media queries responsive
