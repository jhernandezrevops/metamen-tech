# Dashboard Components

Componentes del dashboard:
- `avatar-display.tsx` - Visualización del avatar
- `health-bar.tsx` - Barra de salud (10 corazones)
- `level-banner.tsx` - Banner de nivel actual
- `stats-card.tsx` - Tarjetas de estadísticas
- `vectors-radar.tsx` - Gráfico de radar de vectores
- `time-matrix.tsx` - Matriz de tiempo
- `countdown-timer.tsx` - Temporizador de cuenta regresiva
